# 🍽️ Smart Restaurant Management System

A full-stack restaurant ordering and sales management application built using **FastAPI** and **Streamlit**.
This project allows users to browse menu items, place multiple orders, and track real-time sales.

---

## 🚀 Live Demo  
🔗 https://smart-restaurant-management-system-udnrybhfkwawzzwx84usyt.streamlit.app

## 🚀 Features

* 🛒 Add multiple items to cart
* 💵 Automatic total price calculation
* 📊 Sales tracking system
* ⚡ Fast API backend (FastAPI)
* 🎨 Interactive UI (Streamlit)
* 🔁 Real-time order processing

---

## 🧠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** FastAPI
* **Language:** Python
* **API Server:** Uvicorn

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/Smart-Restaurant-Management-System.git
cd Smart-Restaurant-Management-System
```

---

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

### 3️⃣ Run backend (FastAPI)

```
python -m uvicorn main:app --reload
```

---

### 4️⃣ Run frontend (Streamlit)

```
python -m streamlit run app.py
```

## 🌐 API Endpoints

| Method | Endpoint | Description       |
| ------ | -------- | ----------------- |
| GET    | `/menu`  | Get menu items    |
| POST   | `/order` | Place an order    |
| GET    | `/sales` | View sales report |
